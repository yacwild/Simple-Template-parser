#Simple-Template-parser

Helper supported:

### if 
```
{%if expression%}
 ...
{%elseif expression%}
 ...
{%else%}
 ...
{%endif%}
```
### each
```
{%each data as value,index%}
 ...
{%endeach%}
```
### var
```
{%varname=expression%}
```
### expression
other expression
```
{% true %}
```
